08 Breadcrumbs

Breadcrumbs are a navigational aid commonly used in user interfaces, especially on websites, to show users their current location within a website's hierarchy. They typically appear horizontally across the top of a webpage or below the header, presenting a trail of links that represent the path from the homepage to the current page.

Home / Category / Subcategory / Current Page

HTML Code


        <div class="breadcrumbs serif-font u-fs-h3 l-py-1" style="border: 1px solid var(--clr-accent); max-width: 30em;">
        <a href="#">Home</a> /
        <a href="#">Category</a> / 
        <a href="#">Subcategory</a> / 
        Current Page
    </div>
