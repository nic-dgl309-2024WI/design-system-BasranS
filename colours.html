Text Colours


Black text for body

For background colours the text is black


Hex Code: #0E172

CSS utility class: u-text-body

CSS variable name: --clr-body-text

White text for accent

For accent colours the text is white


Hex Code: #FFFFFE

CSS utility class: u-text-accent

CSS variable name: --clr-accent-text


Colours Information


Colours Use Utility Class Colour Contrast Ratio
Primary Background Used as Main Background bg-primary 16.86:1 PASS
Secondary Background Used as Secondary Background bg-secondary 12.19:1 PASS
Tertiary Background Used to differentiate some sections bg-tertiary 12.33:1 PASS
Text Body Used for Headings and Paragraphs bg-text-body 16.86:1, 12.19:1, 12.33:1 PASS
Primary Accent Used as primary accent colour bg-primary-accent 17.82:1 PASS
Secondary Accent Used as secondary accent colour bg-secondary-accent 7.03:1 PASS
Text accent colour Used as text accent colour bg-text-accent 17.82:1, 7.03:1 PASS
