    Utility Classes

  • serif-font: Heading font
  • sans-font: Body font
  • u-fs-h1: h1 font size
  • u-fs-h2: h2 font size
  • u-fs-h3: h3 font size
  • u-fs-body: Body font size
  • u-fw-bold: Bold font weight
  • u-fw-normal: Normal font weight
  • u-lh-heading: Line height for heading
  • u-lh-body: Line height for body
  • u-letter-spacing: letter spacing
